# PIZZA PLACE

Sales catalog with order form functionality.

## 🚀 Quick Start

1. Open `index.html` in a web browser to view the catalog
2. Product images are in the `assets/` folder
3. Customers can place orders directly through the form

## 📦 Files

- `index.html` - Main catalog page with order form
- `assets/` - Product images (3 files)

## 🌐 Deploy to GitHub Pages

```bash
git add .
git commit -m "Update catalog"
git push origin main
```

Then enable GitHub Pages in your repository settings (Settings → Pages → Deploy from main branch).
